### React Overview

**What is React JS?**

Java Script library used to develope UI (Front End) applications built by Meta(facebook) in 2013.

**Origin:** As the users continue to grow on facebook, pages needs to be reloaded or refreshed in-order to see the 
updates on messages or notifications on the facebook page while scrolling on the facebook page(for normal JS)

**Component based architecture:**
say, whenever we get updates on the parts like notifications or messages, only that part gets re-loaded not everything on the page.
